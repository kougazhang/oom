## 循环
哪种方法生效取决于使用的 bash 解释器，如 `/bin/sh` 还是 `/bin/bash1`, 不同 bash 解释器之间语法上存在细微差异。

方法1：
```shell
for i in {0..5};do echo $i;done
```

方法2:
```shell
for loop in `seq 0 $n`;do echo $i;done
```
