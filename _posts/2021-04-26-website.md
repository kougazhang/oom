## Introduce
+ 本地开发环境: Macos Catalina
+ 参考主题: [cayman](https://github.com/pages-themes/cayman)

## 本地开发环境搭建
1. 升级 ruby 到 3.0. `brew install ruby`
2. 把 [cayman](https://github.com/pages-themes/cayman) clone 到本地.
3. 使用 [Gemfile](https://github.com/kougazhang/kougazhang.github.io/blob/gh-pages/Gemfile) 覆盖原来的文件.
4. 在项目的根目录下安装依赖, `sudo bundle install`
5. 运行本地服务器, `sudo bundle exec jekyll serve`

## 配置介绍
+ 项目文件 [_config.yml](https://github.com/kougazhang/kougazhang.github.io/blob/gh-pages/_config.yml)

### 导航栏模块的元数据

 以根目录下的 `about.md` 为例:

```
# layout 表示渲染后的布局为 Page
layout: page

# 导航栏的名称为 "关于我"
title: 关于我

# 简介
tagline: 不积跬步无以至千里

# md 渲染后生成 /about.html 
permalink: /about.html

# 指向 about.md 这个文件
ref: about

# 导航栏的次序
order: 0
```

### 文章元数据

以该项目自带的文件 `2017-04-20-post-content-styles.md` 为例

```
layout: post
title: "Post content styles"
tagline: "This post demonstrates post content styles"

# 文章链接的 URL, 该文章的链接是: https://<domain>/junk/2017/04/20/post-content-styles.html
categories: junk

# 文章配图
image: /thumbnail-mobile.png

# tags 文章的标签
tags: nsq

# 作者
author: "Bart Simpson"
meta: "Springfield"
```
