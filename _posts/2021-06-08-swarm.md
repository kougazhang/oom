## 安装
+ [官方文档](https://docs.ethswarm.org/docs/installation/install/)
+ 安装系统：Macos

### Bee 的相关依赖
+ Bee Clef, Bee 的签名工具。

### 安装 Bee Clef
```shell
brew tap ethersphere/tap
brew install swarm-clef
```

运行：

```shell
brew services start swarm-clef
```

检查 Bee Clef 是否在运行：

```shell 
launchctl list | grep swarm-clef
```

### 安装 Bee 

```shell
brew tap ethersphere/tap
brew install swarm-bee
```

运行：
```shell
brew services start swarm-bee
```

## 配置 Bee
+ [官方文档-配置](https://docs.ethswarm.org/docs/working-with-bee/configuration/)


|config|type|meaning|
|---|---|---|
|full-node|bool|是否以全节点（full node）启动。全节点会占用大量的网络资源。默认值为 false。|
|swap-endpoint|string|blockchain endpoints, 接入区块链的地址|
|resolver-options|List[string]|resolver-options，接入区块链的集群地址|
|db-open-files-limit|int32|打开 leveldb 时的文件描述符上限|
|debug-api-enable|bool|默认值为 false，是否启动 debug 模式|
|debug-api-addr|string|debug api 地址|

### 配置文件路径
```
vi /usr/local/etc/swarm-bee/bee.yaml
brew services restart swarm-bee
```
