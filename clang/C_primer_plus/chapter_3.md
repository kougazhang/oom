# 数据类型

## 存储单位

最小的存储单位是位(bit), 使用电路对应 0 或 1. 在平常计算数据时, 不会考虑 bit.

字节(byte) 是常用的计算机存储单位, 也是在计算数据量时用到的最小单位. 1 个字节有 8 位, 也就是有 256 种组合. 所以可以使用字节表示数字和字符(比如 ASCII 表), 一般在编程语言中使用单引号表示表示字符, 如 `'a'`

 ## 整数

整数会转化成二进制存储, 如 7 的二进制是 `111`, 在 8 位字节中存储该数字, 需要把前  5 位都设置成 0, 后 3 位设置成 1.

## 浮点数

浮点数会以 e 记数法的方式进行存储: 符号 + 小数 + 指数.

## int 类型

// 创建存储空间
